# Tutorials

Learning-oriented walkthroughs that teach you Tentacles by building something real. Start here if you are new — every concept is introduced in the order you need it, with runnable code at each step.

## What you will build

| Tutorial | Stack | Time | You will learn |
|---|---|---|---|
| [Your first model](/tutorials/your-first-model) | `@kbml-tentacles/core` | 20 min | Contracts, models, instances, queries |
| [A React todo app](/tutorials/react-todo-app) | `core` + `react` | 30 min | `<View>`, `<Each>`, `useModel`, props wiring |
| [A Vue todo app](/tutorials/vue-todo-app) | `core` + `vue` | 30 min | Vue adapter, `emit` routing, composables |
| [A Solid todo app](/tutorials/solid-todo-app) | `core` + `solid` | 30 min | Accessor-based props, `<For>` integration |

## Before you start

Each tutorial assumes:

- **Node.js ≥ 20** and a package manager (npm, yarn, or pnpm).
- **Basic TypeScript** — tutorials use generics and type inference heavily.
- **A little effector knowledge** helps but is not required — stores and events are explained as they come up.

You do **not** need to read the [Explanation](/explanation/) section before doing the tutorials. It exists to deepen your mental model *after* you have working code.
